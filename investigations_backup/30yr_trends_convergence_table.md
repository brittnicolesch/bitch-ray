# The Extraction Stack: How Media, Pharma, Finance, Surveillance, and Platforms Hollowed the American Middle Class (1996–2026)

From 1996 to 2026, American decline was not one plot but a convergence of extraction systems: deregulated media, financialized labor, deindustrialization, pharmaceutical dependency, surveillance law, dark-money politics, data-broker capitalism, and engagement-optimized platforms. These systems did not require a single command center. Their incentives aligned like gears. The result was a middle class stripped economically, fragmented socially, tracked digitally, and governed increasingly through private infrastructure outside democratic oversight.

### Causal Grading Scale
Every data point in this stack is tagged with a causal grade to define the exact nature of its implementation.

| Grade | Meaning | Example Application |
| :--- | :--- | :--- |
| `[Documented Fact]` | Event/law/company/action occurred exactly as stated. | The PATRIOT Act expanded surveillance authority. |
| `[Enabled]` | Created the legal/structural conditions for later abuse. | Telecom deregulation enabled media consolidation. |
| `[Incentivized]` | Profit/governance structure predictably pushed behavior. | Algorithmic ranking incentivized outrage and polarization. |
| `[Exploited]` | Actors intentionally used an existing, organic trend. | Political campaigns exploited platform targeting. |
| `[Deliberately Engineered]` | Intent + funding + mechanism + outcome are proven. | Purdue Pharma's aggressive opioid marketing campaigns. |
| `[Hypothesis]` | Highly plausible theory but requires further primary sourcing. | Data-control may permanently replace older physical kompromat models. |
